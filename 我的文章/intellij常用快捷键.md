# intellij常用快捷键

ctrl + d 复制光标所在行

Ctrl + Tab 切换文件

ctrl + w 根据语法只能选中一块代码

ctrl + y 删除光标所在的行

ctrl + r 搜索选中的区域

ctrl + j 查看代码生成所有模板

Ctrl + Shift + Space 智能补全代码

Alt + Insert 生成构造函数、get、set、toString等方法

Ctrl + Alt + L 格式化代码

输入public static void main的首字母：psvm  然后按tab或者enter，就会写好main方法   

System.out.print(); 键入：sout

输入： fori
