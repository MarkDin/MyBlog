>
> https://api.ulearning.cn/studyrecord/studyingstatus

返回数据:

```json
{
  "status" : 0,
  "courseName" : null,
  "chapterName" : null,
  "sectionName" : null
}
```

> GET 方法
>
>https://api.ulearning.cn/studyrecord/initialize/[课程ID]

```json

```
